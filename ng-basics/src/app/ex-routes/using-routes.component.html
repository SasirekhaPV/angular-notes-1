Using Routes

steps to use routes:

* set the base href <base href="/">

* import and use RouterModule

* navigation of component require a placeholder. Use of router-outlet directive to show component according route state.

* for load multiple components side by side we use named outlets.

* define routes array using Routes (app-routing.module.ts)

* use of routes with parameters. 'ex-routes/:key'

* we can access route parameters using ActivatedRoute (using-routes.default.component.ts)

* use navigate method from Router to go to some other route.
